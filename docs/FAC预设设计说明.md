## 格式说明

### Main 参数映射

> ⚠️ **勘误 (2026-05-15)**: 旧版本文档 Main 3-5 映射有误。以下映射已根据 C++ 源码 `DfxDspPreset.cpp` 中的 `#define` 常量修正。详细说明见 `docs/游戏FAC调音指南.md` §1.2。

| Main | 效果 | 范围 | 说明 |
|------|------|------|------|
| Main 0 | Fidelity | 0-127 | 清晰度/保真度，提升中高频细节 |
| Main 1 | Surround | 0-127 | 环绕声/空间感，拓宽声场 |
| Main 2 | (保留) | 0-127 | 未使用 |
| Main 3 | **Ambience** | 0-127 | 氛围感/混响，增加空间反射 |
| Main 4 | **DynamicBoost** | 0-127 | 动态压缩，让小声音更突出 |
| Main 5 | **BassBoost** | 0-127 | 低音增强，增强低频力度 |

> **注意**: Flutter 启动引擎时会通过 `--fidelity/--surround/--ambience/--dynamic-boost/--bass` 命令行参数覆盖 Main 0/1/4/5 的值（0-10范围）。因此 fac 文件中的 Main 值主要作为 fallback 默认值，**真正生效的是 EQ 频段设置**。

### EQ 频段定义
| Band | CF (Hz) | 频率范围 | 声学意义 |
|------|---------|---------|---------|
| 1 | 62.5 | 超低频 | 爆炸/载具/直升机轰鸣 |
| 2 | 128.75 | 低频 | 脚步声基础/低频噪音 |
| 3 | 238.311 | 中低频 | 脚步声核心/男声 |
| 4 | 444 | 中频 | 人声/脚步声辨识度 |
| 5 | 805 | 中频 | 脚步清晰度/枪声主体 |
| 6 | 1360.79 | 中高频 | 脚步细节/女声/金属声 |
| 7 | 2519.84 | 高频 | 方向感/空间定位 |
| 8 | 4400.12 | 高频 | 枪声清脆度/齿音 |
| 9 | 7930.48 | 极高频 | 刺耳声/空气感 |
| 10 | 12570 | 极高频 | 泛音/高频噪声 |

## 各游戏预设设计

### 1. apex.fac — Apex Legends
**参考**: 1.fac (apex)  
**场景**: 大逃杀，混合距离战斗，需要兼顾远近脚步声+团队沟通

| 参数 | 值 | 设计意图 |
|------|-----|---------|
| Main 0 (Fidelity) | 35 | 适度清晰度 |
| Main 1 (Surround) | 13 | 轻微空间感 |
| Main 4 (Ambience) | 38 | 轻度环境增强 |
| Main 5 (DynamicBoost) | 0 | 无动态压缩，保持自然 |

**EQ 策略**:
- Band 1-3: 轻微增强 (0~+1) — 保持低频自然
- Band 4: -3 — 削减中频浑浊
- Band 5: +1 — 轻微脚步增强
- Band 7: -3 — 控制高频刺耳
- Band 8: +2 — 增强枪声清脆度
- Band 9: -5 — 大幅削减极高频刺耳
- **特点**: 均衡型，远近兼顾，不突兀

### 2. cs2.fac — Counter-Strike 2
**参考**: 3.fac (csgo2)  
**场景**: 紧凑室内地图，极度依赖精确脚步定位

| 参数 | 值 | 设计意图 |
|------|-----|---------|
| Main 0 (Fidelity) | 25 | 适中清晰度 |
| Main 1 (Surround) | 13 | 轻微空间感 |
| Main 4 (Ambience) | 51 | 中度环境增强帮助定位 |
| Main 5 (DynamicBoost) | 25 | 轻度动态压缩 |

**EQ 策略**:
- Band 1: +2 — 轻微低频补偿
- Band 3: +4 — 增强中低频脚步
- Band 4: -3 — 削减中频浑浊
- Band 5: +1 — 轻微中频增强
- Band 7: -3 — 控制高频
- Band 8: +1 — 保持枪声清晰
- Band 9: +7 — 大幅提升极高频，增强材质分辨（金属/混凝土差异）
- Band 10: -1 — 轻微削减
- **特点**: 中低频+极高频双增强，脚步声材质辨识度最高

### 3. pubg.fac — PUBG
**参考**: 4.fac (pubg2)  
**场景**: 8km 大地图，草地/木板/水泥地多样地形，需要远距离脚步+枪声抑制

| 参数 | 值 | 设计意图 |
|------|-----|---------|
| Main 0 (Fidelity) | 0 | 关闭保真度，避免过度锐化 |
| Main 1 (Surround) | 0 | 关闭环绕，减少失真 |
| Main 4 (Ambience) | 64 | 中高度环境增强，帮助远距离定位 |
| Main 5 (DynamicBoost) | 0 | 无动态压缩 |

**EQ 策略**:
- Band 2: -1 — 轻微削减低频噪音
- Band 3: -8 — **大幅削减中低频**，消除载具/轰炸区轰鸣干扰
- Band 4: -3 — 削减中频浑浊
- Band 5: +3 — 增强脚步声核心频段
- Band 6: +12 — **强中高频提升**，脚步声清晰度最大化
- Band 7: +3 — 增强方向感
- Band 8-10: -5 — 统一削减高频，降低枪声刺耳度和环境噪声
- **特点**: 极端"V型"EQ，削低频+提中高频，脚步声最突出，长时间对战不疲劳

### 4. valorant.fac — 无畏契约
**参考**: 3.fac (csgo2) 改良  
**场景**: 5v5 室内战术射击，混凝土/金属地面，技能音效丰富

| 参数 | 值 | 设计意图 |
|------|-----|---------|
| Main 0 (Fidelity) | 28 | 适中清晰度 |
| Main 1 (Surround) | 12 | 轻微空间感，分辨上下层 |
| Main 4 (Ambience) | 42 | 中度环境增强 |
| Main 5 (DynamicBoost) | 18 | 轻度动态压缩 |

**EQ 策略**:
- Band 1-2: -4/-2 — 削减低频，消除室内混响轰鸣
- Band 3: +3 — 增强中低频脚步
- Band 4: +5 — 增强中频脚步核心
- Band 5: +7 — **强中频增强**，脚步声最清晰
- Band 6: +5 — 增强中高频细节
- Band 7: +2 — 保持方向感
- Band 8: -1 — 轻微控制枪声刺耳
- Band 9-10: -3/-2 — 削减极高频，降低技能音效尖锐度
- **特点**: 中频峰值型，脚步声突出但不刺耳，技能音效柔和

### 5. deltaforce.fac — 三角洲行动
**基于理解原创设计**  
**场景**: 大战场+室内CQB混合，载具/直升机/爆炸等大动态，需要自然不突兀

| 参数 | 值 | 设计意图 |
|------|-----|---------|
| Main 0 (Fidelity) | 32 | 适度清晰度 |
| Main 1 (Surround) | 18 | 中度空间感，大战场定位 |
| Main 4 (Ambience) | 48 | 中度环境增强 |
| Main 5 (DynamicBoost) | 22 | 轻度动态压缩，平衡大动态 |

**EQ 策略**:
- Band 1: -4 — 削减超低频轰鸣（直升机/载具）
- Band 2: -2 — 削减低频噪音
- Band 3: +3 — 增强中低频脚步
- Band 4: +5 — 增强中频脚步核心
- Band 5: +6 — 增强中频清晰度
- Band 6: +5 — 增强中高频细节
- Band 7: +1 — 保持自然方向感
- Band 8: -2 — 控制枪声刺耳
- Band 9-10: -3/-2 — 削减极高频
- **特点**: 均衡偏中频，兼顾大战场定位+室内CQB，效果自然不突兀

## 后续调整建议

1. **主观听感验证**: 每个预设需要在实际游戏中测试 15-30 分钟，确认：
   - 脚步声是否足够清晰
   - 枪声是否过于刺耳或沉闷
   - 长时间使用是否疲劳

2. **参数微调方向**:
   - 脚步声不够 → 提升 Band 5 (805Hz) 或 Band 6 (1360Hz)
   - 枪声太刺耳 → 降低 Band 8 (4400Hz) 或 Band 9 (7930Hz)
   - 低频轰鸣 → 降低 Band 1 (62.5Hz) 或 Band 2 (128Hz)
   - 方向感差 → 提升 Main 1 (Surround) 或 Band 7 (2520Hz)

3. **与 UI 默认值同步**: 当前 `preset_service.dart` 中的 0-10 默认值与 fac 文件中的 0-127 默认值差异较大，建议统一。
